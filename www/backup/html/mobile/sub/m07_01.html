찾아오시는 길

서울재활병원 찾아오시는 길

  • 주소 : 서울시 은평구 갈현로 11길 30 (구, 은평구 구산동 191-1)
  • 대표번호 : 6020-3000, 팩스 : 6020-3015

교통안내

버스

  • 파란버스 - 702번 / 751번, 초록버스 - 7022번 → '시립서북병원' 정류장

지하철

  • 6호선 응암역 ②번 출구로 나와 진행 방향 반대로 돌아 리더스아파트 끼고 우회전하여 60미터 전방에서 버스 승차

마을버스

  • 마을버스 2번 은평천사원 정류장 앞

자가용

  • 상암월드컵운동장 → 서울기독대학교 방면
  • 강북지역에서 방문할 경우 : 내무순환도로(성산방향)
  • 잠실(강남)에서 방문할 경우 : 잠실대교 → 강변북로(일산방향)
  • 일산에서 방문할 경우 : 자유로(잠실방향)

주차안내

  • 엔젤스헤이븐 주차장 이용 (병원 우측 서부재활체육센터 옆, 지도 참고)
  • 병원 이용객 최초 1시간 무료
  • 1시간 이후 30분당 500원 부과
  • 병동 : 입퇴원 당일 무료 주차, 장기주차는 금합니다.
  • 낮병동 : 낮병동 이용기간 중 무료 주차권을 발급해 드립니다.
